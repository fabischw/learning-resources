| EASY         | Ctrl + Z                      | undo                                 |                 |
| EASY         | Ctrl + Y                      | redo                                 |                 |
| EASY         | Win + R                       | open program / folder                |                 |

| EASY         | Win + L                       | sign out                             |                 |
